# 🚀 CareerPilot AI

An AI-powered Resume Intelligence Platform that helps job seekers analyze resumes, improve ATS scores, prepare for interviews, and generate career documents using Large Language Models.

## ✨ Features

- 📄 Resume Upload & PDF Parsing
- 🤖 AI Resume Analysis
- 🎯 ATS Score Analysis
- 💼 Career Recommendations
- 🎤 Interview Question Generator
- 📄 Cover Letter Generator
- 🔗 LinkedIn Summary Generator
- 📊 Job Description Matcher
- 🤖 AI Career Assistant

## 🛠️ Tech Stack

- Python
- Streamlit
- Groq API (Llama 3.3)
- PyPDF2
- ReportLab
- HTML
- CSS
